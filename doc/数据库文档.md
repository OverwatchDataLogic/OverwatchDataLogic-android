# 数据库文档

## 赛事表(Sports)

|       字段       |  类型  |       描述       |
| :--------------: | :----: | :--------------: |
|      Title       | String |     赛事名称     |
|    FullTitle     | String |     赛事全称     |
|   EnglishTitle   | String |   赛事英文名称   |
| EndlishFullTitle | String |   赛事英文全称   |
|   Description    | String |     赛事介绍     |
|       Logo       | String |    赛事 Logo     |
|    StartDate     |  Date  |   赛事开始时间   |
|     EndDate      |  Date  |   赛事结束时间   |
|      About       | String |     赛事相关     |
|      Prize       | Number | 比赛奖金(人民币) |

---

## 赛程表(schedules)

|   字段    |      类型       |   描述   |
| :-------: | :-------------: | :------: |
| StartDate |      Date       | 开始时间 |
|  EndDate  |      Date       | 结束时间 |
|  Stages   | Array\<Object\> | 赛程阶段 |

* Stage: 普通赛事一般只有一个赛程阶段

### 赛程阶段表(Stages)

|  字段   |      类型       |   描述   |
| :-----: | :-------------: | :------: |
|  Name   |     String      | 赛程名称 |
| Enabled |      Bool       | 是否启用 |
| Matches | Array\<Object\> |   赛次   |

#### 赛次表(Matches)

|        字段        |       类型       |   描述   |
| :----------------: | :--------------: | :------: |
|    Competitors     | Array\<Object\>  |   对手   |
|       Scores       |   Array\<Int\>   |   比分   |
|  ConclusionValue   |       Int        |  决胜值  |
| ConclusionStrategy |      String      | 决胜策略 |
|       Winner       |      Object      |  获胜者  |
|       State        |      String      | 比赛状态 |
|       Games        |      Array       | 每局比赛 |
|    DateCreated     |       Date       | 创建时间 |
|     StartDate      |       Date       | 开始时间 |
|      EndDate       |       Date       | 结束时间 |
|        Wins        |   Array\<Int\>   | 胜者比分 |
|        Ties        |   Array\<Int\>   | 关系比分 |
|       Losses       |   Array\<Int\>   | 败者比分 |
|       Videos       | Array \<Object\> |   视频   |

* Scores: e. `[{"value": 1},{"value": 3}]`
* ConclusionValue: e. 4
* ConclusionStrategy: e. MINIMUM
* State: PENDING/CONCLUDED
* Wins: e. `[1,3]`
* Ties: e. `[0,0]`
* Losses: e. `[3,1]`

##### 对手/胜者(Competitors/Winner)

|      字段       |  类型  |   描述   |
| :-------------: | :----: | :------: |
|      Name       | String | 队伍名称 |
|  HomeLocation   | String |   城市   |
|  PrimaryColor   | String |  主色调  |
| SecondaryColor  | String |  次色调  |
| AbbreviatedName | String |   简称   |
| AddressCountry  | String |   国家   |
|      Logo       | String |   Logo   |
|      Icon       | String |   Icon   |
| SecondaryPhoto  | String |  次图片  |

##### 每局比赛(Games)

|   字段   |      类型       |   描述   |
| :------: | :-------------: | :------: |
|  Number  |       Int       |   场次   |
|  Points  |  Array\<Int\>   |   比分   |
|   Map    |     String      |   地图   |
| MapScore | Array\<Object\> | 地图比分 |
|  State   |     String      | 比赛状态 |

* Points: e. `[0, 1]`
* MapScore: e.`[{"team1": 0,"team2": 1}]`

##### 视频(Videos)

|     字段     |  类型  |      描述      |
| :----------: | :----: | :------------: |
|    Number    |  Int   |      场次      |
| Description  | String |      描述      |
|   VodLink    | String |    视频地址    |
| ThumbnailUrl | String | 视频缩略图地址 |

## 队伍表(Teams)

|      字段       |      类型       |      描述      |
| :-------------: | :-------------: | :------------: |
|      Name       |     String      |    队伍名称    |
| AbbreviatedName |     String      |    队伍简称    |
|  HomeLocation   |     String      | 城市 (or 线上) |
|    Accounts     | Array\<Object\> |    媒体账号    |
|  PrimaryColor   |     String      |     主色调     |
| SecondaryColor  |     String      |     次色调     |
| AddressCountry  |     String      |      国家      |
|   Description   |     String      |    队伍介绍    |
|      Match      |     String      |    比赛经历    |
|   CreateTime    |      Date       |    成立时间    |
|   IsDissolved   |      Bool       |   是否已解散   |
|  DissolvedTime  |      Date       |    解散时间    |
|     Honour      |     String      |    主要荣誉    |

### 媒体账号(Accounts)

|    字段     |  类型  |   描述    |

| AccountType | String | 媒体类型  |
| AccountLogo | String | 媒体 Logo |
|    Value    | String | 媒体链接  |
